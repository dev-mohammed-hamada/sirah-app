---
description: Game mechanics rules for hearts, XP, stars, streaks, and progression
---

## Hearts

- 5 per session, -1 per wrong answer, 0 = must retry
- Refill: 1 heart per 30 min or full refill on new day

## XP

- Awarded per correct answer. Never resets.
- Bonuses: Perfect stage +20, Speed +5, First Try +10

## Stars

- 1-50% score -> 1 star, 51-80% -> 2 stars, 81-100% -> 3 stars
- Only highest score kept per stage

## Streak

- Requires >= 1 completed stage or daily challenge per day
- Resets on miss. Milestones: 3, 7, 14, 30 days

## Daily Challenge

- 1 question per day from previously learned content
- Counts toward streak

## Unlock

- 24/30 stars in a stage group unlocks the next group

## Stage Flow

1. Narrator welcome -> 2. Panel+question -> 3. Panel+question -> 4. Panel (conclusion) -> 5. Final quiz (3-5 Qs) -> 6. Results -> 7. Narrator closing + teaser

- Max 3 sentences per panel, simple Arabic
- No question type repeated >2x in a row
