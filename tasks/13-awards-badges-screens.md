# Task 13: Awards + Badges Screens

## Priority: LOW (V1 prototypes)

## Description

Trophy room showing XP level, achievement stats, and badges. V1 has 6 prototype badges with locked placeholders.

## Screens

### Awards Screen (design 17)

- Night Sky gradient (premium museum feel)
- XP Level: circular progress ring (120px), level number, XP to next level
- Stats Grid (2x2): Total Stars, Stages Completed, Longest Streak, Perfect Runs
- Badges Section: 3-column grid, earned (golden border + shimmer) vs locked (gray + "?")
- Tap earned badge → detail overlay
- "View all" → Badges Screen

### Badges Screen (design 18)

- Category tabs: All, Streak, Perfection, Speed, Stages, Goals
- Badge grid (3 columns): Earned, Locked, In-Progress states
- Badge detail modal on tap

## V1 Badges

1. Beginner — complete first stage
2. Persistent — 7-day streak
3. Star — earn 15 stars
4. Guardian — link with father
5. Master — complete all 10 stages
6. Hero — earn 30 stars
   7-10. Locked placeholders

## Acceptance Criteria

- [ ] XP level ring with progress
- [ ] Stats grid with count-up animations
- [ ] Badge grid with 3 states
- [ ] Category tab filtering
- [ ] Badge detail modal
- [ ] Locked badges shake on tap
